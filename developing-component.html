用于开发组件

有时候我们并不想执行页面渲染，而是用Vanex来开发一个组件，这时，还是可以使用start API，只要不传如container值，就会返回一个React Component。 

import { start } from 'vanex';

const Comp = start({
  component: CountComponent,
  models: {
    // 注册模型
    countModel,
  },
});
